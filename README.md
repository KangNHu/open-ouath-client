# open-ouath-client

#### 介绍
##### 一个第三方OAuth2协议的授权框架
- 内置支持的第三方登录
  -  giee
  -  知乎
  -  qq
  -  新浪微博
  -  微信
  -  faceboot
  -  cas (耶鲁大学开源SSO项目 )
  -  google 
- 支持用户自定义扩展第三方登录方式
- 支持自定义第三方token和业务系统token绑定
